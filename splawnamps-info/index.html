Web technology is a fast-moving field, so fast at times that many find it really difficult to keep their websites up-to-date with the moving user demands of responsive layouts and a smartphone-first approach.

Motivations / Goals

  • Design a low-cost, static, responsive website
  • Flexible layout handling based on client viewport
  • Decrease overall website load time
  • Keep feature parity w/ old site
  • Improve checkout UX
  • Improve on SEO

Tools

  • The static site generation toolchain is based on Wintersmith, a minimalistic static site generator for NodeJS
  • The site setup uses:
    • Bootstrap for responsive element setup & handling (try out different viewport sizes!)
    • JQuery for small interactive elements
    • Pug (formerly Jade) template engine for partials & mixins
    • Example hosted on BitBalloon
