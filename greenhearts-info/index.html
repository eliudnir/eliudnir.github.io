Our locally grown food delivery service have asked for help to renew and update their web presence. Their main goals were to create a responsive, smartphone friendly website that retains their SEO and simplifies their ordering/checkout system. I threw my hat in the ring.

Motivations / Goals

  • Create a repeatable workflow to generate a low-cost hosted website that meets the clients needs and can easilly be extended
  • Design a simple, clean & updated UI that scales well across all common device families
  • Keep feature parity while simplifying order process / checkout system
  • Retain all SEO work from original site
  • Continue to support client in future IT & Web needs

Tools

  • UI layer is VueJS view framework together with the Vue Router
  • Responsive magic and styling thanks to SemanticUI
  • Reproducible build output and dev server by Webpack and BabelJS (w/ loaders)
  • Example hosted on BitBalloon
